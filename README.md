A mini message board that allows users to view and submit new messages. It includes population of a PostgreSQL database.

This project demonstrates an understanding of basic Express concepts, including routers, controllers, views, and error handling, as well as understanding of populating and using a PostgreSQL database.

Environment variables include:

- `PORT`
- `PGHOST`
- `PGUSER`
- `PGDATABASE`
- `PGPASSWORD`
- `PGPORT`

See [env.example](https://github.com/deepend15/odin-mini-message-board/blob/main/.env.example) for more information.

Live link: https://odin-mini-message-board-production-a955.up.railway.app/
